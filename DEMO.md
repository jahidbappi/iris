# Iris Demo Video Script

Record a 60–90 second demo for your portfolio and README.

## Setup

1. Run `npm run dev` in `iris/`
2. Open http://localhost:3000
3. Use demo mode (no API keys) or add keys for live generation

## Shot list

| Time | Action |
|------|--------|
| 0:00 | Landing page — highlight "See. Speak. Create." |
| 0:10 | Click **Open Studio** |
| 0:15 | Enable camera, show vision panel |
| 0:25 | Push-to-talk: "Generate a minimalist poster with deep indigo gradients" |
| 0:40 | Show thinking state → canvas result → timeline events |
| 0:55 | Open **Observability** — show cost and latency metrics |
| 1:05 | Click **Share Replay**, paste link in new tab |

## Tips

- Use a dark room for visual polish
- Keep browser at 1440px width
- Export as MP4, upload to README or LinkedIn

## Tools

- macOS: QuickTime Screen Recording
- Free: OBS Studio
